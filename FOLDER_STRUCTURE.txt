TECH STACK
════════════════════════════════════════════════════════════════════

⚛️  Framework:          Next.js 14 (App Router)
📘 Language:            TypeScript 5.5

════════════════════════════════════════════════════════════════════

npm run build           → Build for production
npm run start           → Start production server
npm run lint            → Run ESLint


════════════════════════════════════════════════════════════════════
ENVIRONMENT VARIABLES NEEDED
════════════════════════════════════════════════════════════════════

✅ POSTGRES_URL                  → Vercel Postgres connection
✅ POSTGRES_PRISMA_URL          → Prisma connection
✅ POSTGRES_URL_NO_SSL          → Non-SSL connection
✅ POSTGRES_URL_NON_POOLING     → Non-pooling connection
✅ POSTGRES_USER                → Database user
✅ POSTGRES_HOST                → Database host
✅ POSTGRES_PASSWORD            → Database password
✅ POSTGRES_DATABASE            → Database name
✅ RESEND_API_KEY               → Resend email API key
✅ EMAIL_FROM                   → Sender email address
   NEXT_PUBLIC_APP_URL          → Application URL (optional)

WHAT'S NEXT?
════════════════════════════════════════════════════════════════════

1. Install Node.js (if needed)
2. cd seershub-landing
3. npm install
4. Set up Vercel Postgres (see SETUP.md)
5. Set up Resend API (see SETUP.md)
6. Create .env.local file
7. npm run dev
8. Open http://localhost:3000
9. Test the waitlist
10. Customize & deploy!
